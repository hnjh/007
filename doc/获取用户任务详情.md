获取用户的任务
--------

Request:

```
GET v1/users/{id}/tasks/{task_id}
```

Response:

```
HTTP 200
{
  "msg":"SUCCESS!",
  "data": {"task_id":xxx,
           "status":1,
           "result":"xxx",
           "create_time":"xxx"
           }
}
```

```
HTTP 400
{
  "msg"  :"BADARG!",
  "data" : null
}
```
